# m4a-mp3-converter-py 

#### simple m4a => mp3 converter w/ gui using ffmpeg developed with python 3.


## Install
* `git clone https://github.com/revzim/m4a-mp3-converter-py`
* `pip3 install -r requirements.txt`
* `python/python3.X main.py`
